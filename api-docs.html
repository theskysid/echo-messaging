üí¨ Message Endpoints

GET
/api/messages/public

Description: Retrieve all public chat messages

Authentication: Required (Bearer Token)

GET
/api/messages/private?user1=alice&user2=bob

Description: Retrieve private messages between two users

Authentication: Required (Bearer Token)

Query Parameters:

  • user1 - First user's username
  • user2 - Second user's username

üîå WebSocket Integration

WebSocket URL: ws://localhost:8080/ws

Protocol: SockJS + STOMP

Subscribe Destinations:

  • /topic/public - Public chat messages
  • /user/{username}/queue/messages - Private messages

Send Destinations:

  • /app/chat.addUser - Join chat
  • /app/chat.sendMessage - Send public message
  • /app/chat.sendPrivateMessage - Send private message

JavaScript Example:

const socket = new SockJS('http://localhost:8080/ws');
const stompClient = Stomp.over(socket);

stompClient.connect({
  'Authorization': 'Bearer ' + token
}, function(frame) {
  // Subscribe to public messages
  stompClient.subscribe('/topic/public', function(message) {
    const chatMessage = JSON.parse(message.body);
    console.log('Public message:', chatMessage);
  });
  
  // Send a message
  stompClient.send('/app/chat.sendMessage', {}, JSON.stringify({
    sender: 'username',
    content: 'Hello everyone!',
    type: 'CHAT'
  }));
});
